# Fall_Detection

多執行緒影像擷取：使用獨立執行緒持續讀取攝影機畫面，避免主執行緒被阻塞

即時 FPS 顯示：

畫面左上角綠色文字顯示 FPS

終端機每秒輸出一次 FPS 數值

流暢退出機制：按 q 鍵可安全釋放資源並關閉視窗

解析度設定：預設 640x480 以平衡畫質與效能
